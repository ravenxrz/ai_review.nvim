*ai_review.nvim* AI/Codex Git hunk review sidebar

===============================================================================
CONTENTS                                                        *ai_review*

Commands:
  :AiReviewOpen       Open the sidebar
  :AiReviewClose      Close the sidebar
  :AiReviewToggle     Toggle the sidebar
  :AiReviewRefresh    Refresh the sidebar

Default sidebar mappings:
  p       show conflict-style preview for current hunk
  ]g      next hunk and preview
  [g      previous hunk and preview
  a/s     accept hunk
  x/r     reject hunk
  u       undo accepted/rejected hunk
  R       refresh and clear processed rows
  zR      expand all
  zM      collapse all
  S       toggle submodule scan
  q       close sidebar

Setup:
>
  require('ai_review').setup({
    sidebar = { side = 'left', width = 50, winfixbuf = true },
  })
<


===============================================================================
CONFIGURATION                                           *ai_review-configuration*

Default setup:
>
  require('ai_review').setup({
    sidebar = { side = 'left', width = 50, winfixbuf = true },
    git = { root_cache = true, max_diff_lines = 20000 },
    submodules = {
      enabled = true,
      recursive = true,
      max_depth = nil,
      include_untracked = true,
      max_untracked_files = 200,
      max_untracked_file_size = 256 * 1024,
    },
    scanner = {
      async = true,
      concurrency = 8,
      render_debounce_ms = 80,
      git_timeout_ms = 5000,
    },
  })
<
